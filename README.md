# Dictation Cleanup Spike

A lightweight Chrome Extension designed to sanitize real-time dictation by intercepting commands and performing conservative text cleanup

## Setup Instructions

1. Clone the repository: git clone https://github.com/A-Rafayy/Dictation-Cleanup-Spike.git

2. Open Chrome and navigate to chrome://extensions/

3. Enable developer mode.

4. Click Load Unpacked and select the cloned folder

5. You should see the "Dictation Cleanup Spike" card appear in your extension list.
